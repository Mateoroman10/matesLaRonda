# La Ronda — Mates y Tradición 

Sitio web oficial de **La Ronda**, un emprendimiento argentino dedicado a la venta de mates, bombillas, termos y accesorios materos artesanales de calidad.

## ¿De qué se trata?

La Ronda nace de la pasión por el mate y la cultura que lo rodea. Este sitio es la vidriera digital del emprendimiento: un espacio donde los clientes pueden conocer los productos, ver precios y ponerse en contacto para hacer pedidos o consultas.

La propuesta apunta a quienes valoran lo artesanal, lo auténtico y el ritual del mate compartido.

## ¿Qué tiene el sitio?

- **Carrusel de fotos** en la portada con imágenes del ambiente matero
- **Catálogo de productos** con fotos, descripción y precios: mates, bombillas, termos, bolsos y sets completos
- **Video** mostrando el arte de cebar mate
- **Reseñas** de clientes reales de todo el país
- **Formulario de contacto** para pedidos y consultas, integrado con Formspree
